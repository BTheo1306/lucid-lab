## 4. Stratégie de contenu — Calendrier

### Fréquence
- **3 posts par semaine** minimum pour que l'algorithme LinkedIn vous favorise
- Idéalement : Mardi, Jeudi, Vendredi

### Types de contenus (rotation)

| Type | Fréquence | Exemples |
|------|-----------|---------|
| Cas client / résultat | 1×/semaine | "On a automatisé X chez Y → -3h/jour" |
| Tutoriel / Comment faire | 1×/semaine | "Comment qualifier 200 leads avec n8n en 1h" |
| Opinion / Réflexion | 1×/semaine | "Pourquoi les consultants IA ne livrent pas" |
| Derrière les coulisses | 2×/mois | Build en cours, outil qu'on utilise |
| Question / Sondage | 1×/mois | Engagement facile |

### Format optimal LinkedIn
- **Hook** : 1ère ligne percutante (visible sans "Voir plus")
- **Corps** : Bullet points, chiffres concrets, pas de jargon
- **CTA** : 1 seul, à la fin
- **Hashtags** : 3-5 max, pertinents (#IA #Automatisation #AgentsIA)
- **Image/vidéo** : Toujours — le reach est 2-3× supérieur

### Hashtags principaux à suivre et utiliser
```
#intelligenceartificielle #automatisation #agentsIA #workflow
#nocode #n8n #crm #startup #pme #operationsexcellence
#agentic #openai #anthropic #LLM
```
