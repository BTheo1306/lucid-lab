Ce qu'on construit ensemble

Trois modules complémentaires pour structurer la présence digitale de BSP 37, générer des leads qualifiés pour l'activité film solaire et doter Sophia d'un assistant IA privé calé sur son activité.

01

Objectif

BSP 37 est un groupement familial ancré dans l'artisanat depuis 1989 : bâche, sellerie, store, impression numérique et personnalisation textile. L'accompagnement cible trois axes :

  • Refonte du site web — remplacer le site Wix actuel par une version moderne, rapide et bien référencée, sans frais d'entrée : tout est inclus dans le mensuel.
  • Acquisition film solaire via Meta Ads — ouvrir un nouveau canal de génération de leads sur Facebook / Instagram pour l'activité d'installation de film solaire, avec production de vidéos courtes pour les campagnes.
  • Système IA Claude + Obsidian — installer sur la machine de Sophia un système de connaissance privé pour centraliser processus, fournisseurs, catalogue et suivi client.

Module 01 — Site web (bsp37.com)

  • Audit du site Wix existant et recommandations SEO.
  • Refonte complète : performance, référencement naturel, contrôle total sans abonnement Wix.
  • Pages : accueil, bâche / sellerie / store, impression numérique, objets personnalisés, contact.
  • Galerie de réalisations et catalogue produits intégrés.
  • Déploiement et configuration du nom de domaine bsp37.com.
  • Maintenance mensuelle incluse : mises à jour, corrections, suivi des performances.

Module 02 — Meta Ads / Film solaire

  • Création et configuration du compte Meta Ads (Facebook / Instagram).
  • Ciblage local : zone géographique, audiences et centres d'intérêt.
  • Création d'une landing page dédiée à la génération de leads film solaire.
  • Gestion mensuelle : optimisation, A/B tests visuels, reporting mensuel.
  • Production de vidéos courtes pour les campagnes (Reels / Stories) : 150 € HT / vidéo.
  • Suivi des leads : formulaire dédié + tableau de bord partagé (traçabilité UTM).
  • Commission de performance : 2 % HT sur les devis signés issus directement des leads Meta Ads, déclarés mensuellement.
  • Budget publicitaire Meta non inclus — recommandation de départ : 200–400 € / mois, géré directement par Sophia.

Module 03 — Système IA Claude + Obsidian

  • Installation et configuration d'Obsidian + Claude Code sur la machine de Sophia.
  • Création d'un vault BSP 37 dédié : processus, fournisseurs, catalogue, réalisations, suivi clients.
  • Configuration des sources approuvées et session d'ingestion initiale.
  • Formation à la routine hebdomadaire d'utilisation.
  • Remise du guide d'installation et du guide d'utilisation Lucid-Lab (PDF).
